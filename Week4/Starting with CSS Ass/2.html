What is css?
CSS is the language we use to style a Web page.

HTML was created to describe the content of a web page, like: h1 This is a heading /h1 p This is a paragraph. /p When tags like font , and color attributes were added to the HTML 3.2 specification, it started a nightmare for web developers. Development of large websites, where fonts and color information were added to every single page, became a long and expensive process.
